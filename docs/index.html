About app-name

Usage

This section explains how to import app-name and call its templates in your XSL application or library.

Import app-name

To import app-name.xsl into another XSL file, the following handworks are needed. Since app-name depends on some xslet libraries, they are needed to import, too.

  1. Add the namespace declarations of xslbook and xslutil in a XSL file.

    <xsl-stylesheet version="1.0"
      xmlns:xsl="http://www.w3.org/1999/XSL/Transform"
      xmlns:ut="https://github.com/xslet/2020/xslutil"
      xmlns:book="https://github.com/xslet/2020/xslbook"> <!-- This! -->
      ...

  2. Add a import element of app-name in the XSL file.

      ...
      <!-- dependency XSL files are already imported in app-name.xsl -->
      <xsl:import href="./app-name.xsl"/>
      ...
